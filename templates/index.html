多参仪全功能控制平台接口说明

一.功能项管理

1.添加功能项

向接口 /api/function POST json数据就可以添加功能项，json格式是这样的

{ "serial_number":"1001", "function_type":0, "function_name":"心电测量" }

2.获取所有功能项

GET接口 /api/function 就可以获取所有功能项

二.license管理

1.添加license

向接口 /api/license POST json数据就可以添加license，json格式是这样的

{ "functions":[ {"function_number":"1001","function_name":"心电图"}, {"function_number":"1002","function_name":"血糖"}, {"function_number":"1003","function_name":"血脂"} ], "amount":3 }

2.获取所有license

GET接口 /api/license 就可以获取所有功能项

三.功能配置管理

1.添加功能配置

向接口 /api/config POST json数据就可以添加功能项，json格式是这样的 { "serial_number":"1001", "function_type":0, "function_name":"心电测量" }

2.获取所有功能配置

GET接口 /api/config 就可以获取所有功能项

四.license校验，有两种方式

1.发送GET请求

GET方式调用接口 /api/verify url后传递license和sn参数即可 如：/api/verify?license=DFJT-TDRO-Q4I3-W7CU&sn=123456abcdef

2.发送POST请求

向接口 /api/verify POST json数据即可，json格式是这样子

{ "license":"DFJT-TDRO-Q4I3-W7CU", "sn":"1234567zbc" }
